# 🎥 Guion Video 4 – Preparando el Entorno de Desarrollo

## 1. Introduccióntu
- Breve saludo.  
- Explicar que en este video vamos a preparar el **entorno de trabajo**.  
- Motivar: *“Antes de empezar a programar nuestro coche, necesitamos configurar bien las herramientas. Esto nos permitirá trabajar de forma profesional y comunicarnos con el microcontrolador ATmega sin problemas.”*  

---

## 2. Descarga e instalación de VS Code
- Explicar qué es **Visual Studio Code (VS Code)**:  
  - Es un **editor de código moderno, potente y gratuito**.  
  - Lo usaremos como entorno de programación principal.  
- Guiar paso a paso la descarga desde la web oficial.  
- Instalarlo en el ordenador.  

---

## 3. Extensiones necesarias
- Explicar que las **extensiones** amplían las funcionalidades de VS Code.  
- Instalar una por una (mostrar en pantalla cómo se hace):  
  - **PlatformIO IDE** → para programar el microcontrolador y gestionar librerías.  
  - **Aura Theme** → para un entorno visual más cómodo.  
  - **Better Comments** → para organizar y resaltar comentarios en el código.  
  - **Todo Highlight** → para marcar tareas pendientes dentro del código.  

👉 Explicar que estas extensiones no son solo “decoración”:  
- Facilitan la programación.  
- Ayudan a que el código sea más claro y fácil de mantener.  

---

## 4. Git y GitHub (mención breve)
- Explicar muy brevemente:  
  - **Git** es una herramienta para llevar el control de cambios en tu proyecto.  
  - **GitHub** permite guardar tu proyecto en la nube y colaborar con otros.  
- No dar una clase completa, solo remarcar que:  
  - Es muy útil para profesionales y estudiantes.  
  - También es recomendable para aficionados, porque guarda un respaldo de tu trabajo.  
  - Gracias a los servidores de Microsoft (GitHub), tendrás tu proyecto seguro y accesible desde cualquier sitio.  

👉 Mostrar en pantalla cómo instalar Git y crear un repositorio de ejemplo (aunque no se explique a fondo).

## 5. Comunicación con el microcontrolador
- Explicar que vamos a usar VS Code + PlatformIO para **comunicarnos con la placa ATmega**.  
- Aclarar que esto es lo que nos permitirá:  
  - Escribir programas en C/C++.  
  - Cargarlos en la placa.  
  - Hacer debugging y pruebas.
